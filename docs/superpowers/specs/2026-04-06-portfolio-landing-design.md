# Portfolio Landing Page — Design Spec

**Date:** 2026-04-06
**Type:** Одностраничный лендинг-портфолио с анимациями

## Обзор

Минималистичный одностраничный сайт-портфолио без секции проектов (будет добавлена позже).
Три секции: Hero → Навыки/Обо мне → Контакты.
Единая тема: тёмная, разработческий стиль с терминальными элементами.

## Архитектура

- **Стек:** Vanilla HTML/CSS/JS (без фреймворков — сайт быстрый, лёгкий, легко поддерживать)
- **Анимации:** CSS transitions + Intersection Observer API (без зависимостей)
- **Canvas-анимации:** Два canvas-элемента в hero — падащий код и пиксели
- **Адаптивность:** Mobile-first CSS, media breakpoints на 768px и 1024px

---

## Секция 1: Hero

### Структура
- Полноэкранная секция (100vh), `position: relative`
- **Фон:** Градиент (#0f0c29 → #302b63 → #24243e)
- **Canvas 1 (подложка):** Падащий Python/Node код, полупрозрачный (opacity ~0.12), monospace шрифт, зелёные символы (#7ee787). Строки медленно падают сверху вниз (Matrix-style, но с реальным кодом).
- **Canvas 2 (пиксели):** Маленькие полупрозрачные квадратики разных цветов, падащие по экрану
- **Контент (overlay, по центру):**
  - Заголовок: крупный текст по центру (например "Full-Stack Developer")
  - Подзаголовок: короткое описание
  - Кнопка CTA: "Смотреть навыки" (scroll вниз)
  - **Фото справа:** круглое фото с двойной рамкой (inner border 3px + outer border 1.5px с glow-эффектом)
  - **Имя под фото:** мелкий текст под рамкой

### Анимации Hero
- Падающий код: canvas, requestAnimationFrame, строки кода перемещаются сверху вниз циклически
- Падащие пиксели: canvas, множество маленьких квадратов с рандомной скоростью/прозрачностью
- При загрузке: fade-in контента (opacity 0→1, ~1s)
- CTA-кнопка: пульсирующий glow

## Секция 2: Навыки + Обо мне

### Структура
- Один терминальный виджет на всю ширину контента
- Заголовок секции: "Навыки & Опыт" — текст по центру, стиль терминальной строки

### Вкладки (tabs)

**Таб 1 — `cat skills.txt`:**
- Два столбца по 3 навыка (grid 2x3)
- Каждый навык: название цветным текстом + прогресс-бар (█-символы) + процент + подстрока с деталями
- Цвета навыков: Python (#7ee787), Frontend (#56b6c2), Node.js (#484f58), AI (#5865F2), DBs (#4299e1), DevOps (#f6b82d)
- Анимация при скролле: fade-in + slide-up

**Таб 2 — `cat about.txt`:**
- Box-рамка (ASCII-style `┌───┐`)
- Текст (структурированный, с цветовым выделением ключевых слов):

```
$ cat about.txt

Андроид-разработчик
Kotlin · Jetpack Compose · Node.js · Python

Разрабатываю Android-приложения на Kotlin + Jetpack Compose,
самостоятельно проектирую REST API на Node.js + Express
и работаю с данными через Python. Связываю клиент, бэкенд
и данные в одну цепочку.

▸ Исследование перед кодом
  Изучаю API, структуру данных и ограничения до начала разработки

▸ Чистая архитектура
  Разделяю клиент и сервер, использую репозитории

▸ Автоматизация
  Парсинг, тестирование, импорт данных — рутина должна работать сама

▸ Ориентация на продукт
  Продукт должен решать задачу пользователя, а не просто работать

Цели: развиваюсь в сторону создания полноценных продуктов — 
от мобильных приложений до серверной инфраструктуры и автоматизации.

_  ← мигающий курсор
```

- Мигающий курсор внизу

### Интерактивность
- Переключение табов кликом на вкладки
- Hover-эффект на навыках (подсветка)
- При скролле в viewport: плавное появление терминала

## Навигация

- Фиксированная навигация сверху (header): логотип/имя + якорные ссылки (Skills, Contact)
